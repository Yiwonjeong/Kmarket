# Kmarket 종합 온라인 쇼핑몰 

### 개발기간
2022.12.05 ~ 2022.12.28 (총 18일)

### 배경 및 목적
- 기존 전자 상거래 플랫폼(이하 온라인 쇼핑몰)이 가지고 있는 불편한 접근성과 복잡성을 개선
- 판매자와 사용자에 친화적인 쇼핑몰 개발에 방점을 둠
- 보다 편리한 온라인 쇼핑 환경에 기여

### 주요 기능
- 회원가입 / 로그인
- 상품등록 / 상품 목록 / 상품 보기 / 장바구니 / 주문하기
- 고객센터 공지사항 / 자주 묻는 질문 /
